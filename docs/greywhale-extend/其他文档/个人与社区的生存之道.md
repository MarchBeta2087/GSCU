# 个人与社区的生存之道

在我看来，个人的生存之道和社区的生存之道，各有七条，如果个人能掌握这七条，个人就能立于不败之地；如果社区能掌握这七条，社区就能持续运转。

## 一个社区要如何生存？

### 1. 拒绝接受商业接管，而非选择完全与商业隔绝

一个社区一旦被商业接管，这个社区就会开始推出各种必须你花真金白银才能买到的特权，并且会逐渐脱离群众。久而久之，许多想要从社区学习知识的成员们就会对这个社区产生厌恶，最后离开这个社区。

然而，如果一个社区只是一味地"用爱发电"，没有任何收入或者入不敷出，注定会因为资金短缺而走向灭亡。

一个比较好的办法，就是寻找可为社区提供硬件等资源赞助的企业，向他们宣传这个社区是做什么的，到底怎么样。当社区获得了赞助之后，社区核心领导层就会有更好的资源来维持社区运转。

### 2. 照顾各位成员与广大群众的生计，坚决打击内部的恶意破坏者，用广大成员与群众的智慧抵御外部攻击

要让成员们愿意在社区做出贡献，社区需要照顾成员们与广大群众的生计。

一个社区如果能向成员们与广大群众提供资金或技术支持，成员们就能在社区成长起来，许多新成员也会加入到社区。

然而，社区内外存在着恶意。社区内部滋生的恶意破坏者，会通过言论或技术手段对社区进行破坏，从而掌控或摧毁社区。

这个时候，社区的核心领导层就应该合理利用技术手段制止内部的恶意破坏者，让他们无法继续行动。实在不行，还可以动用法律手段，将破坏者绳之以法。

当然，外部攻击也值得注意。社区的核心领导层可以合理采纳各位成员乃至广大群众的策略，让社区能够在经历外部攻击之后生存下来。

### 3. 接纳想要加入社区的广大群众，让社区走向成熟

一个社区的发展需要越来越多的成员加入。如果一个社区想要生存下去，那么，社区成员们可以向广大群众们宣传这个社区到底有多好，在这个社区能学到什么，但不能夸大事实。

当一个社区接纳了广大群众，这个社区就会有越来越多的新成员，社区的凝聚力会更强，社区就会走向成熟。

### 4. 社区如果希望长期存在，就需要避免触碰政治与道德上的高风险边界

这一点很重要。如果一个社区触碰了政治红线，社区的成员们会感到恐惧，他们担心自己因为发表了什么言论，哪怕听起来很轻松，第二天就遇到了危险。在这种情况下，成员们会默不作声，社区陷入一片死寂。

更可怕的是，触碰政治红线，很有可能导致社区面临封杀。

另外，一个社区也应该避免触碰道德红线。当一个社区触碰了道德红线，社区内外都会有很多人反感。

在这种情况下，一些成员会选择离开，以免因为与这个社区的联系遇到麻烦。

如果一个社区想要生存，想要留住成员，千万不要触碰政治红线与道德红线！

### 5. 坚决拒绝独裁，并且给予一切成员适当的民主权利，无论成员的年龄、出身、性别等等

一个社区如果由独裁者管理，这个社区的氛围会变得非常压抑，成员们在社区会恐惧讨论，恐惧发言。给社区带来的并不是凝聚力，而是恐惧与死寂。

要让社区生存下去，社区需要给予一切成员适当的民主权利。

是的，一切成员，无论成员的年龄、出身、性别等等。当民主权利被真正赋予一切成员时，他们就能畅谈社区的未来，让社区能够健康成长。

### 6. 回报每一位做出了贡献的成员，让他们能够留在社区，愿意为社区的繁荣与成熟做贡献

一个社区的成长，往往需要成员的支持与贡献。要想让成员们认为这个社区有爱且可靠，社区应该给予相应的回报。

当成员们获得了应有的回报之后，他们就会认为发自内心地爱这个社区，只要这个社区不会出什么大问题，他们愿意为社区的繁荣与成熟做贡献。

### 7. 谁主张，谁举证；没有调查，就没有发言权

如果社区内部发生了矛盾、纠纷，社区要让双方能够陈述矛盾、纠纷的经过，如果其中一方能够证明另一方有过错，社区就应该让过错方的承担责任；如果双方都证明对方有过错，社区可以权衡利弊，合理分配双方责任；如果双方都不能证明对方有过错，社区还可以想办法让双方和解。

另外，任何成员如果要建设这个社区，社区就应该接受成员们的调查，如果一位成员完成了充分的、严谨的调查，他就能为社区建言献策；反之，如果一位成员未经充分的、严谨的调查就发言，大概率是胡说八道、瞎指挥。

## 一个人要如何生存？

### 1. 掌握几项无论到哪都能换饭吃的"硬技能"，自己养活自己，而不是完全指望社区或雇主

有的时候，你的社区或者雇主真的遇到了困难，或者根本没打算回报你什么，这个时候，千万不要指望社区或雇主，而是要自己养活自己。

你可以从另外一个社区，或者从你在街上看到的，学几项"硬技能"换饭吃，比如修电脑、补衣服。

当你掌握了一些"硬技能"，虽然你不一定能靠它们买到一辆迈巴赫，但维持每日三餐应该足够。

### 2. 不要理会巨魔，不要试图与巨魔争辩，更不要尝试拉帮结派。如果接下来可能会与他人发生争吵，有事好商量，协商解决。如果你的人身权利受到侵犯，可以寻求法律援助

巨魔显然让人厌恶，通过言论激怒群众就是他们的目的。他们绝不会因为你的当面指责停止行动，千万不要与他们争辩。

另外，如果你尝试带着你的朋友一起对付巨魔，你很有可能会失败，你的朋友也很有可能离你而去。

如果你不幸与他人发生了争吵，首先要记住的是，有事好商量，私下协商是一个很好的选择。

另外，如果你的人身权利受到侵犯，你可以寻求法律援助，向律师陈述你的经历，你并非只能单打独斗。

### 3. 在帮助别人时，如果你实在没有能力，或者你教不会，你可以寻求擅长相关方面的第三人的帮助。另外，在寻求帮助的时候要有礼貌

帮助他人自然是一件好事。然而，你不是超人，你在帮助他人的同时可能会遇到困难。

在这个时候，如果你实在没有能力，或者你教不会，寻求擅长相关方面的第三人的帮助并不可耻，相反，这是被鼓励甚至被认为是必要的。

当你寻求帮助的时候，请保持礼貌，每个人的耐心都是有限的，只有保持礼貌，你才有可能得到帮助。

### 4. 自觉注意言行，不要触及政治红线和道德红线

有一个成语叫做"祸从口出"。如果你公开发表了政治敏感言论，可能会引起大家的恐慌，而且你也会面临危险。

为了能够在这个社会生存下去，千万不要触碰政治红线，更不要公开发表政治言论。

另外，如果你发表了不当言论，触及道德红线，你会面临大家的谴责，最终，你可能会为此感到羞耻。

### 5. 尽早学会如何在孤身一人的时候从绝境中崛起，有的时候，大家可能没打算帮你，也可能不想帮你

虽然大多数时候，你可以与大家谈论，遇到问题也可以向大家求助。

然而，也许有一天，大家可能都没有帮助你的打算，也可能都不想帮你，你会陷入孤立无援的困境。

所以，趁现在，你应该学会如何在孤身一人的时候从绝境中崛起，真到了那一天，你要靠你自己的智慧生存下去，回归正常生活。

### 6. 尽早远离 996 等不合理制度，而不是完全拒绝工作。记住，你不是为了老板的迈巴赫而工作的，你也不能指望不工作也能让自己生存下去，你是为你自己的生存而工作的

996 确实是令众多工作者（当然，这些职业除外：需要 24 小时持续运转，以应对突发情况的职业，例如医生、护士、交警等等；需要直接为群众服务的职业，例如超市收银员、酒店前台接待员等等，......，反正你肯定不希望你常去的超市在傍晚六点就关门了）厌恶的制度。然而，如果所有人都拒绝工作，这个社会就会停转。

如果你觉得工作时间太长，加班费还很少（或者干脆没有），你要记住，你不能指望不工作也能让自己生存下去，你是为你自己的生存而工作的，尝试按照正常的工作时间工作，比如一天工作八小时，每周工作五天。如果你的公司实在不允许你一天工作八小时，每周工作五天，你可以选择辞职去找下一家公司，实在辞不了或者辞了之后工作更难找，你还可以尝试自己在家工作，比如以独立开发者的身份在 GitHub 上发布你的项目，并利用你的项目获得赞助，当你每天的收入对于你的生活而言绰绰有余，你就可以安然无恙地继续在家为自己工作。

### 7. 信任是有限的。老板的发言不一定是真的，朋友看到的，也不一定是真的，只有你的实际能力是真的

你的老板说"共克时艰"，很有可能首先就是要对你下手，然后找个时机跑路。

你的朋友告诉你去哪里可以每天都赚钱，那里也不一定能赚到很多钱，甚至可能存在各种危险。

信任是有限的，没有一个人会永远跟你讲真话，但你的实际能力是真的。

常识告诉你，三角形是最稳定的形状。但你可能会遇到某篇文章说"四边形是最稳定的"，这明显是错误的，你可以找四根筷子和四条橡皮筋，每根筷子的一端与另一根筷子绑在一起，组成一个四边形，你会发现，只要你稍微触碰其中一角，这个四边形很快就会变形。接下来，你再去找三根筷子和三条橡皮筋，每根筷子的一端与另一根筷子绑在一起，组成一个三角形，你会发现，你触碰任何一角，你都无法让这个三角形变形。

常识告诉你，四加四等于八。但你可能会遇到某篇文章说"四加四等于九"，你应该很快就能知道这是错误的，你可以拿出一张纸，左边画四个圆，右边画四个圆，然后数数看一共有几个圆，你很容易就能数出是八个，而不是九个。

记住，你的实际能力，永远是你最可信的伙伴。

## 总结

我撰写这篇博文，为的就是帮助个人和社区在风云变幻的社会上生存下去。如果适当参照上述的个人七条，结合个人的智慧，个人就能立于不败之地；如果适当参照上述的社区七条，结合社区成员们和广大群众的智慧，社区就能持续运转。
