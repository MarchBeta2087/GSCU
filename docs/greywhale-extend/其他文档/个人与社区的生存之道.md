# 个人与社区的生存之道

在我看来，个人的生存之道和社区的生存之道，各有七条，如果个人能掌握这七条，个人就能立于不败之地；如果社区能掌握这七条，社区就能持续运转。

## 一个社区要如何生存？

### 1. 拒绝接受商业接管，而非选择完全与商业隔绝

一个社区一旦被商业接管，这个社区就会开始推出各种必须你花真金白银才能买到的特权，并且会逐渐脱离群众。久而久之，许多想要从社区学习知识的成员们就会对这个社区产生厌恶，最后离开这个社区。

然而，如果一个社区只是一味地"用爱发电"，没有任何收入或者入不敷出，注定会因为资金短缺而走向灭亡。

一个比较好的办法，就是寻找可为社区提供硬件等资源赞助的企业，向他们宣传这个社区是做什么的，到底怎么样。当社区获得了赞助之后，社区核心领导层就会有更好的资源来维持社区运转。

### 2. 照顾各位成员与广大群众的生计，坚决打击内部的恶意破坏者，用广大成员与群众的智慧抵御外部攻击

要让成员们愿意在社区做出贡献，社区需要照顾成员们与广大群众的生计。

一个社区如果能向成员们与广大群众提供资金或技术支持，成员们就能在社区成长起来，许多新成员也会加入到社区。

然而，社区内外存在着恶意。社区内部滋生的恶意破坏者，会通过言论或技术手段对社区进行破坏，从而掌控或摧毁社区。

这个时候，社区的核心领导层就应该合理利用技术手段制止内部的恶意破坏者，让他们无法继续行动。实在不行，还可以动用法律手段，将破坏者绳之以法。

当然，外部攻击也值得注意。社区的核心领导层可以合理采纳各位成员乃至广大群众的策略，让社区能够在经历外部攻击之后生存下来。

### 3. 接纳想要加入社区的广大群众，让社区走向成熟

一个社区的发展需要越来越多的成员加入。如果一个社区想要生存下去，那么，社区成员们可以向广大群众们宣传这个社区到底有多好，在这个社区能学到什么，但不能夸大事实。

当一个社区接纳了广大群众，这个社区就会有越来越多的新成员，社区的凝聚力会更强，社区就会走向成熟。

### 4. 不要触及政治红线与道德红线

这一点很重要。如果一个社区触碰了政治红线，社区的成员们会感到恐惧，他们担心自己因为发表了什么言论，哪怕听起来很轻松，第二天就遇到了危险。在这种情况下，成员们会默不作声，社区陷入一片死寂。

更可怕的是，触碰政治红线，很有可能导致社区面临封杀。

另外，一个社区也应该避免触碰道德红线。当一个社区触碰了道德红线，社区内外都会有很多人反感。

在这种情况下，一些成员会选择离开，以免因为与这个社区的联系遇到麻烦。

如果一个社区想要生存，想要留住成员，千万不要触碰政治红线与道德红线！

### 5. 坚决拒绝独裁，并且给予一切成员适当的民主权利，无论成员的年龄、出身、性别等等

一个社区如果由独裁者管理，这个社区的氛围会变得非常压抑，成员们在社区会恐惧讨论，恐惧发言。给社区带来的并不是凝聚力，而是恐惧与死寂。

要让社区生存下去，社区需要给予一切成员适当的民主权利。

是的，一切成员，无论成员的年龄、出身、性别等等。当民主权利被真正赋予一切成员时，他们就能畅谈社区的未来，让社区能够健康成长。

### 6. 回报每一位做出了贡献的成员，让他们能够留在社区，愿意为社区的繁荣与成熟做贡献

## 一个人要如何生存？

### 1. 掌握几项无论到哪都能换饭吃的"硬技能"，自己养活自己，而不是完全指望社区或雇主

### 2. 不要理会巨魔，不要试图与巨魔争辩，更不要尝试拉帮结派。如果接下来可能会与他人发生争吵，有事好商量，协商解决。如果你的人身权利受到侵犯，实在不行可以寻求法律援助

### 3. 在帮助别人时，如果你实在没有能力，或者你教不会，你可以寻求擅长相关方面的第三人的帮助。另外，在寻求帮助的时候要有礼貌

### 4. 自觉注意言行，不要触及政治红线和道德红线

### 5. 尽早学会如何在孤身一人的时候从绝境中崛起，有的时候，大家可能没打算帮你，也可能不想帮你

### 6. 尽早远离 996 等不合理制度，而不是完全拒绝工作。记住，你不是为了老板的迈巴赫而工作的，你也不能指望不工作也能让自己生存下去，你是为你自己的生存而工作的

### 7. 信任是有限的。老板的发言不一定是真的，朋友看到的，也不一定是真的，只有你的想法和实际能力是真的
