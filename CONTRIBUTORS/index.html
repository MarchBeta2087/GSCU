    贡献指南 - GSCU 设定集

    欢迎来到 GSCU（全球亚文化俱乐部联盟设定集） 的贡献指南！

    无论你是写作新人、插画师还是世界观狂热爱好者，这里都有你的舞台。

    我们鼓励大家自由创作，但也希望作品能融入整体世界观，因此请阅读以下指南。

    📌 1. 贡献类型

    我们接受多种类型的贡献：

    类型 描述
    📝 设定条目 补充新的俱乐部、队伍、据点、事件或人物档案
    🔧 修正/扩展 修正已有条目或为其补充细节
    🎨 插图/素材 为条目提供插图、地图、照片、截图等
    🌐 翻译 提供中文/英文等语言版本
    💡 点子/悬案 提交灵感、设定漏洞、悬而未决事件等

    小提示：每一个 Issue 或 PR 都是世界的一部分，你的创作即是档案。

    🧰 2. 新手上手步骤

    1. Fork 仓库
      点击右上角 Fork，在你的账户下创建副本。

    2. 选择目标条目或模板
      我们提供模板文件：

    3. docs/模板/俱乐部模板.md（俱乐部）
    4. docs/模板/队模板.md（队伍）
    5. docs/模板/据点模板.md（据点）

    6. docs/模板/研究员模板.md（研究员）

    7. 填写模板

    8. 按照模板格式填写必要信息
    9. 避免直接删除已有内容，尽量添加新的字段

    10. 每个条目保持 150–500 字（可适当扩展）

    11. 提交 PR

    12. Branch 命名示例：add_club_[代号]_[俱乐部名]
    13. PR 描述示例：
      • 新增条目：俱乐部 LSC "光影棋社"
      • 类型：俱乐部
      • 作者：@你的 GitHub 账号
      • 简介：描述俱乐部背景和主要活动

    14. 如果这是修正或扩展已有条目，请在 PR 里说明修改点

    15. 关联 Issue（可选）
      如果你的贡献基于一个灵感或讨论，最好在 PR 中关联对应 Issue

    📌 3. Issue 模板建议

    在提交 Issue 时，请注明：

    • 类型（灵感 / 修正 / 征集 / 讨论）
    • 标题（简洁描述内容，如 "新增 AI 俱乐部档案"）
    • 详细说明：
    • 设定概要或问题描述
    • 预期贡献类型（新条目 / 扩展 / 插图 / 其他）
    • 可选：参考编号、关联条目、灵感来源

    🗂 4. 世界观分层

    为了保证世界观统一，同时鼓励自由创作，我们分层管理：

    • Canon（正典）
      官方认可、必须合并前统一文风的条目
    • Apocrypha（衍生档案）
      社区创作的非主流档案，可自由发挥

      PR 可标注 apocrypha 目录或在标题注明

    🎨 5. 写作风格指南

    • 伪档案体：条目尽量以官方文档、档案、日志、研究记录的风格书写
    • 第一/第三人称都可，但保持一致
    • 避免解释全部真相：留白或悬念是世界观的一部分
    • 编号系统：如涉及 GPC 编号，请遵循现有规则，若新增条目可占用未分配编号

    🌟 6. 贡献等级与荣誉

    贡献者可获得不同类型的认可：

    • 轻量参与：小修小补、插图、注释 → 列入 CONTRIBUTORS.md
    • 中度参与：新增条目或事件 → Featured Contributor + 可能合并至正典
    • 深度参与：撰写关键历史事件、世界观扩展 → 可能成为 Canon Contributor

    ⚠️ 7. 注意事项

    • 尊重他人创作，不随意修改他人 PR 内容
    • 避免完全重复已有条目
    • 如果你希望加入自己原创角色或组织，请尽量标注作者信息
    • 所有内容需符合社区规范，不含敏感或违法信息

    📌 8. 交流与反馈

    • Discussions：参与世界观辩论、灵感交换、悬案讨论
    • Issues：报告问题、提交创作建议
    • PR：提交条目、修正或扩展

    GSCU 是一个开放创作的平台，你的每一次提交都在扩充这个世界。
    欢迎大胆创作，但也请遵守以上流程，以便所有贡献都能被归档和保存。

    感谢你参与 GSCU 世界观建设！

    我们期待看到你的创意档案加入全球亚文化俱乐部联盟的历史记录中。
